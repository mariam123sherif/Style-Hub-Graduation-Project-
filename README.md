## 📖 Project Overview

**Style Hub** is a full-stack e-commerce platform designed to deliver a seamless and stylish online shopping experience. Built by a collaborative development team, Style Hub focuses on clean UI, smooth user flows, and a powerful admin backend to manage a growing product catalog.

---

## ✨ Features

### 🔍 Product Browsing & Search
Explore a wide range of products with intuitive filtering, sorting, and a fast search experience to help users find exactly what they're looking for.

### 🔐 User Authentication
Secure sign-up, login, and session management — keeping customer accounts safe and personal data protected.

### 🛒 Shopping Cart & Wishlist
Users can add items to their cart or save favorites to a wishlist for a convenient and personalized shopping experience.

### ⚙️ Admin Dashboard
A dedicated admin panel for managing products, orders, and users — giving the team full control over the platform.

### ⭐ Product Reviews & Ratings
Customers can leave reviews and star ratings on products, helping the community make informed purchasing decisions.

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18+)
- [MongoDB](https://www.mongodb.com/)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/style-hub.git

# Navigate to the project directory
cd style-hub

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env

# Start the development server
npm run dev
```
